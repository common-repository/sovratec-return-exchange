Manage WooCommerce customer order return and exchange request.

== Description ==

Provide Return & Exchange order management features for WooCommerce.
This plugin allows WooCommerce store owners track and management return and exchange orders.  

<h3>Key Features</h3>

The plugin is packaged with the following features:

* Customers initiate return & exchange flow by submitting requests.  
* Return & exchange requests go through workflow with store owner’s approval.  
* Each step is tracked with date is recorded.  The plugin offers both customer perspective and store owner perspective.  
* Customer can check return/exchange status without additional phone call or email.  
* Store owner can manage return/exchange more effectively by tracking every step happed during return/exchange.

Install and activate the plugin.
After installation, Return & Exchange displays on the right hand.  That’s the link for store owner.  
For your customers, once they log in, Return & Exchange link will be in their account view.
